# 📝 Journal des Modifications (CHANGELOG)

Toutes les modifications notables apportées à **The Uprising Screenrecorder** seront documentées dans ce fichier.

## [1.2.0] - 2026-03-01

### Ajouté

- **Incrustation WebCam (PiP)** : Overlay caméra circulaire intégré directement dans l'enregistrement.
- **Sous-titres par IA** : Gravure automatique des sous-titres dans la vidéo lors de l'export.
- **Mixage Audio Dual** : Capture simultanée et mixage de l'audio système et du micro.
- **Contrôle de Vitesse** : Vitesse de lecture variable (0.5x à 4x) préservée à l'export.
- **Tableau de Bord** : Nouvel écran d'accueil haute fidélité inspiré par Adobe Premiere.
- **Release Automation** : Workflow GitHub Actions pour des releases automatisées.

### Amélioré

- **Performance d'Export** : File d'attente optimisée selon la puissance de la machine.
- **UI/UX** : Nouvelles icônes Lucide et animations de composants Radix.
- **Accessibilité** : Ajout de labels ARIA et de titres sur tous les éléments interactifs.

### Corrigé

- **WebM Duration** : Correction des méta-données de durée lors des enregistrements longs.
- **Types TypeScript** : Résolution de plusieurs erreurs `any` et variables inutilisées.
