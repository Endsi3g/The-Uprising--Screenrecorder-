# 🎥 The Uprising Screenrecorder

Un enregistreur d'écran pro-grade et un hub créatif, conçu pour les créateurs de contenu exigeants. Build ultra-rapide avec Electron, React et PixiJS.

## 📱 Connexion Mobile (Tableau de Bord)

Vous pouvez accéder à vos idées de vidéos et à vos notes depuis votre téléphone :
1. Assurez-vous que votre téléphone et votre ordinateur sont sur le **même réseau Wi-Fi**.
2. Dans l'application, l'adresse IP de votre serveur sera affichée (ex: `http://192.168.1.15:3001`).
3. Ouvrez le navigateur de votre téléphone et entrez cette adresse.
4. Planifiez vos vidéos en déplacement !

---

## 🔥 Fonctionnalités Clés

- **Enregistrement Haute Performance** : Capture fluide avec accélération matérielle.
- **WebCam PiP (Picture-in-Picture)** : Incrustation caméra circulaire et miroité en temps réel.
- **Sous-titres Auto-Générés (IA)** : Génération automatique de sous-titres optimisée pour le Français.
- **Mixage Audio Dual** : Capturez l'audio système et votre micro simultanément.
- **Éditeur Intégré** : Coupez, zoomez et annotez vos clips sans quitter l'app.
- **Automatisation GitHub** : Déploiements et releases automatisés via CI/CD.

## 🛠️ Installation

```bash
# Installer les dépendances
npm install

# Lancer en mode développement
npm run dev

# Compiler pour Windows/macOS
npm run build
```
